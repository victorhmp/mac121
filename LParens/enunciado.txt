Exercício 1.3.9 de Algs4

Faça o Exercício 1.3.9 de Algs4 (livro; veja abaixo). Para gerar instâncias para seu programa, use a variante abaixo de RandomExpression.java do Web Exercise 2.3.29 de IntroCS:

http://introcs.cs.princeton.edu/java/23recursion/

Chame seu programa de LParens.java. A entrada de seu programa deve ser dada na entrada padrão e a saída deve ser enviada para a saída padrão.

Exemplo de execução:

$ java-introcs LParens
1 + 2 ) * 3 - 4 ) * 5 - 6 ) ) )
[Crtl D]
( ( 1 + 2 ) * ( ( 3 - 4 ) * ( 5 - 6 ) ) )
